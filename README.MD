# Sliding Tiles Puzzles - A* solver

## Aluno: Daniel da Rocha Carneiro

### Implementação do jogo de blocos deslizantes (Jogo dos 8 ou Jogo dos 15) com interface gráfica e solucionador baseado em algoritmo de busca A* e heurística de Manhattan.\

___

## Bibliotecas necessárias

* Numpy

* Pygame

* Copy

## Execução

Basta navegar até a pasta do repositório e executar o seguinte comando:

    python SlidePuzzleUI.py

## Parametros

Os seguintes parâmetros podem ser configurados através das constantes no início do arquivo SlidePuzzleUI.py:

    BOARDWIDTH = 4  #Tamanho do tabuleiro
    ANIMSPEED = 6 #Velocidade da animaç?o de resoluç?o
    NUMBEROFSCRAMBLES = 60 #Número de movimentos aleatórios a partir de um estado inicial para embaralhamento do tabuleiro
